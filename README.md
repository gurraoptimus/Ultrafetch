## ✨ Features

- ⚡ Ultra-fast system info: OS, Kernel, Uptime, Shell, Terminal, Packages
- 💾 Resource monitoring: Memory, Disk, Battery
- 🌍 Network & live weather info
- 📦 Update checking: APT, Pacman, DNF, Zypper
- 🔄 Self-update prompt with animated progress bar
- 🛡️ Robust error handling & clickable OSC 8 hyperlinks

## Installation

```bash
git clone https://github.com/gurraoptimus/Ultrafetch.git
cd ultrafetch
chmod +x ultrafetch.sh
./ultrafetch.sh
```

## Example Output

user@host 1.0 version
