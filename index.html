Ultra Fetch

A modern Linux system overview script inspired by Neofetch and Fastfetch. It displays the core machine status you care about, including OS, memory, disk, CPU usage, GPU status, network info, weather, and update state.

About Ultra Fetch

Ultra Fetch is built to give Linux users a visually clean, information-dense snapshot of machine health and environment details in one command. It is useful for local development, home servers, demos, and quick support diagnostics.

What It Shows

Ultra Fetch combines classic terminal aesthetics with practical status metrics.

System Identity

OS, kernel, hostname, shell, terminal, and package count.

Performance Snapshot

CPU model and usage, GPU model and status, RAM and disk utilization.

Runtime Context

Local IP, uptime, battery health, weather summary, and update info.

How To Setup Ultra Fetch On Linux

Follow these steps to download, install, and run Ultra Fetch on most Linux distributions.

  1. 1. Download the script
    curl -fsSL https://raw.githubusercontent.com/gurraoptimus/Ultrafetch/main/ultrafetch.sh -o ultrafetch.sh
  2. 2. Make it executable
    chmod +x ultrafetch.sh
  3. 3. Run it
    ./ultrafetch.sh
  4. 4. Optional: Install globally
    sudo mv ultrafetch.sh /usr/local/bin/ultrafetch ultrafetch
